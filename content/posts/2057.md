# Go 模拟
 
> [原题 2057.值相等的最小索引](https://leetcode.cn/problems/smallest-index-with-equal-value)
```go
func smallestEqual(nums []int) int {
	for i, num := range nums {
		if i%10 == num {
			return i
		}
	}
	return -1
}
```

# Go 模拟
 
> [原题 2011.执行操作后的变量值](https://leetcode.cn/problems/final-value-of-variable-after-performing-operations)
```go
func finalValueAfterOperations(operations []string) (ans int) {
	for _, op := range operations {
		if op[1] == '+' {
			ans++
		} else {
			ans--
		}
	}
	return
}
```

另一种写法

```go
func finalValueAfterOperations(operations []string) (ans int) {
	for _, op := range operations {
		ans += 44 - int(op[1])
	}
	return
}
```

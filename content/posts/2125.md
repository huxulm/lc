# 按照题意模拟：一次遍历
 
> [原题 2125.银行中的激光束数量](https://leetcode.cn/problems/number-of-laser-beams-in-a-bank)
```go
func numberOfBeams(bank []string) (ans int) {
	pre := 0
	for _, row := range bank {
		cnt := strings.Count(row, "1")
		if cnt > 0 {
			ans += pre * cnt
			pre = cnt
		}
	}
	return
}
```

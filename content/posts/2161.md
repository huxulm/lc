# 划分成三组然后合并
 
> [原题 2161.根据给定数字划分数组](https://leetcode.cn/problems/partition-array-according-to-given-pivot)
```go
func pivotArray(nums []int, pivot int) []int {
	var a, b, c []int
	for _, v := range nums {
		if v < pivot {
			a = append(a, v)
		} else if v == pivot {
			b = append(b, v)
		} else {
			c = append(c, v)
		}
	}
	return append(append(a, b...), c...)
}
```

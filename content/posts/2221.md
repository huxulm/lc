# 简洁模拟 + O(1) 空间
 
> [原题 2221.数组的三角和](https://leetcode.cn/problems/find-triangular-sum-of-an-array)
```go
func triangularSum(a []int) int {
	for n := len(a) - 1; n > 0; n-- {
		for i := 0; i < n; i++ {
			a[i] = (a[i] + a[i+1]) % 10
		}
	}
	return a[0]
}
```

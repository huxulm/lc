# Go 简单模拟
 
> [原题 1910.删除一个字符串中所有出现的给定子字符串](https://leetcode.cn/problems/remove-all-occurrences-of-a-substring)
```go
func removeOccurrences(s, part string) string {
	for {
		i := strings.Index(s, part)
		if i < 0 {
			return s
		}
		s = s[:i] + s[i+len(part):]
	}
}
```

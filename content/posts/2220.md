# Go 一行
 
> [原题 2220.转换数字的最少位翻转次数](https://leetcode.cn/problems/minimum-bit-flips-to-convert-number)
返回两个数异或和的二进制中的 $1$ 的个数。

```go
func minBitFlips(start, goal int) int {
	return bits.OnesCount(uint(start ^ goal))
}
```

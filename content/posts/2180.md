# 模拟 + 位运算
 
> [原题 2180.统计各位数字之和为偶数的整数个数](https://leetcode.cn/problems/count-integers-with-even-digit-sum)
```go
func countEven(n int) (ans int) {
	for i := 1; i <= n; i++ {
		s := 0
		for x := i; x > 0; x /= 10 {
			s ^= x % 10 & 1 // 用异或代替加法
		}
		ans += s ^ 1 // 偶数就 +1
	}
	return
}
```

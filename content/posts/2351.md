# 位运算 O(1) 空间
 
> [原题 2351.第一个出现两次的字母](https://leetcode.cn/problems/first-letter-to-appear-twice)
```go
func repeatedCharacter(s string) byte {
	mask := 0
	for _, c := range s {
		if mask>>(c&31)&1 > 0 {
			return byte(c)
		}
		mask |= 1 << (c & 31)
	}
	return 0
}
```

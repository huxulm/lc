# 简洁写法，自身递归
 
> [原题 2331.计算布尔二叉树的值](https://leetcode.cn/problems/evaluate-boolean-binary-tree)
```go
func evaluateTree(root *TreeNode) bool {
	if root.Left == nil {
		return root.Val == 1
	}
	if root.Val == 2 {
		return evaluateTree(root.Left) || evaluateTree(root.Right)
	}
	return evaluateTree(root.Left) && evaluateTree(root.Right)
}
```

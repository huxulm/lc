# Go 库函数模拟
 
> [原题 2129.将标题首字母大写](https://leetcode.cn/problems/capitalize-the-title)
```go
func capitalizeTitle(title string) string {
	a := strings.Split(title, " ")
	for i, s := range a {
		a[i] = strings.ToLower(s)
		if len(s) > 2 {
			a[i] = strings.Title(a[i])
		}
	}
	return strings.Join(a, " ")
}
```

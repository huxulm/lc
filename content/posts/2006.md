# O(n) 做法
 
> [原题 2006.差的绝对值为 K 的数对数目](https://leetcode.cn/problems/count-number-of-pairs-with-absolute-difference-k)
```go
func countKDifference(nums []int, k int) (ans int) {
	cnt := map[int]int{}
	for _, v := range nums {
		cnt[v]++
	}
	for _, v := range nums {
		ans += cnt[v-k]
	}
	return
}
```

# Go 模拟（避免运算中出现浮点数）
 
> [原题 2303.计算应缴税款总额](https://leetcode.cn/problems/calculate-amount-paid-in-taxes)
把浮点运算放在最后做。

```go
func calculateTax(brackets [][]int, income int) float64 {
	ans, pre := 0, 0
	for _, b := range brackets {
		up, p := b[0], b[1]
		if income <= up {
			ans += (income - pre) * p
			break
		}
		ans += (up - pre) * p
		pre = up
	}
	return float64(ans) / 100
}
```

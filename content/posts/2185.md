# 模拟
 
> [原题 2185.统计包含给定前缀的字符串](https://leetcode.cn/problems/counting-words-with-a-given-prefix)
```go
func prefixCount(words []string, pref string) (ans int) {
	for _, word := range words {
		if strings.HasPrefix(word, pref) {
			ans++
		}
	}
	return
}
```

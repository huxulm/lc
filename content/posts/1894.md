# Go 模拟
 
> [原题 1894.找到需要补充粉笔的学生编号](https://leetcode.cn/problems/find-the-student-that-will-replace-the-chalk)
```go
func chalkReplacer(a []int, k int) int {
	s := 0
	for _, v := range a {
		s += v
	}
	k %= s
	for i, v := range a {
		if k < v {
			return i
		}
		k -= v
	}
	panic(-1)
}
```

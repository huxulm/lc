# Go 贪心
 
> [原题 1877.数组中最大数对和的最小值](https://leetcode.cn/problems/minimize-maximum-pair-sum-in-array)
```go
func minPairSum(a []int) (ans int) {
	sort.Ints(a)
	n := len(a)
	for i, v := range a[:n/2] {
		ans = max(ans, v+a[n-1-i])
	}
	return
}

func max(a, b int) int {
	if a > b {
		return a
	}
	return b
}
```

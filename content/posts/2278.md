# 库函数模拟
 
> [原题 2278.字母在字符串中的百分比](https://leetcode.cn/problems/percentage-of-letter-in-string)
```go
func percentageLetter(s string, letter byte) int {
	return strings.Count(s, string(letter)) * 100 / len(s)
}
```

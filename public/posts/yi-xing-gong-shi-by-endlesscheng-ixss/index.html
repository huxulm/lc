原题 2413.最小偶倍数 视频讲解 已出炉，欢迎点赞三连，在评论区分享你对这场周赛的看法~

根据题意，当 $n$ 为奇数时，答案为 $2n$，当 $n$ 为偶数时，答案为 $n$。

因此答案为

$$ (n\bmod 2 + 1) \cdot n $$

class Solution:
    def smallestEvenMultiple(self, n: int) -> int:
        return (n % 2 + 1) * n

func smallestEvenMultiple(n int) int {
	return (n%2 + 1) * n
}

注：Python 也可以这样写

class Solution:
    def smallestEvenMultiple(self, n: int) -> int:
        return lcm(n, 2)

复杂度分析

  • 时间复杂度：$O(1)$。
  • 空间复杂度：$O(1)$。
